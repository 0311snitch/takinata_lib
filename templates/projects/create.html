What is project?

A project is an abstraction that will contain categories that in turn contain your tasks.

For what?

Tasks from different areas of life can be confusing and difficult to control and organize all the information. The categories in which tasks could be stored also do not completely solve the problem, because the categories "Music" and "Scientific work" on one screen can interfere with each other. To do this, there is an abstraction "Project", which in turn contains categories and allows thematically or in the sense of separate categories from each other.

Example of use

- Project "Business", which contains all the tasks related to the opening and operation of the business.
- Project "University", where you can store all your homework, compound tasks for scientific papers and other tasks.
- The project "Learning English", which will contain all the tasks related to the study of a foreign language.
